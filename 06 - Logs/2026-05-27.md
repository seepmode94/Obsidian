## Para Não Me Perder
*De dias anteriores linkados acima (secção "Para Não Me Perder"):*

```dataviewjs
const cur = dv.current().file;
const links = cur.outlinks.values.filter(l => /^\d{4}-\d{2}-\d{2}$/.test(l.fileName()));
if (links.length === 0) {
  dv.paragraph("_Sem ligação a logs anteriores._");
} else {
  let total = 0;
  for (const link of links.sort((a,b) => b.fileName().localeCompare(a.fileName()))) {
    const page = dv.page(link.path);
    if (!page) continue;
    const pending = page.file.tasks
      .where(t => !t.completed && t.text.trim().length > 0 && t.header.subpath === "Para Não Me Perder");
    if (pending.length > 0) {
      dv.paragraph(`De [[${page.file.name}]]:`);
      dv.taskList(pending, false);
      total += pending.length;
    }
  }
  if (total === 0) dv.paragraph("_Nada em aberto nos linkados._");
}
```
