# LinkedIn — posicionamento profissional

> Data: 2026-05-28
> Contexto: como me apresentar no LinkedIn como **Junior Software Engineer** com responsabilidade client-facing e workflow AI-native, sem inflar título.

---

## Princípios

1. **Assumir "Junior" / "Associate"** — esconder ou inflar é sempre pior. Recrutadores filtram por cargo; aparecer como "Senior" sem o ser tira-me das listas certas e queima credibilidade quando descobrem.
2. **Diferenciação pelo escopo, não pelo título** — client-facing como júnior é raro e vale a pena destacar.
3. **AI-native funciona melhor em júnior do que em sénior** — é sinal geracional, não claim de autoridade.
4. **Ofício primeiro, IA como modificador.** "Mobile Engineer · AI-augmented" ✅. "AI-augmented professional" ❌.

---

## Convenções AI no LinkedIn (2025-2026)

### Funcionam (por ordem de credibilidade)

| Convenção | Quando usar | Exemplo |
|---|---|---|
| **AI-augmented** + ofício | Mais seguro, transversal | *AI-augmented Mobile Engineer* |
| **AI-native** | Funciona bem para júnior/mid | *AI-native developer building with Claude Code + MCP* |
| **AI Engineer** | Só se construo features de IA em produto | *AI Engineer · Flutter + LLM integrations* |
| **Building with [stack]** | Concreto, sem hype | *Mobile engineer · Flutter, Claude Code, MCP* |
| **Agentic** | Só com prova real em produção | *Building agentic workflows for [domínio]* |

### Não funcionam

- ❌ **"Prompt Engineer"** isolado — dated, só tem peso com pedigree (Anthropic, OpenAI).
- ❌ **"AI-first"** sem ofício a seguir — vazio.
- ❌ **"AI evangelist / guru / thought leader"** — repele engenheiros.
- ❌ **Empilhar buzzwords** — "AI-first · agentic · prompt-engineered · GPT-powered" lê-se como bot.
- ❌ **"Vibe coding"** — meme polarizante, fica em tweets.

## Opção A — Headline LinkedIn

Opções seguras e credíveis para o meu caso:

> *Junior Software Engineer @ Seepmode · Flutter + SuiteCRM · client-facing*

> *Junior Full-Stack Developer · Flutter, Node, SuiteCRM · AI-native workflow (Claude Code, MCP)*

> *Associate Software Engineer @ Seepmode · Mobile + CRM integrations · talks to clients, ships features*

Nota: *Associate* é equivalente a júnior mas soa mais profissional internacionalmente.
