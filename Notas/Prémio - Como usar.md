# Prémio — Como usar o valor

Resumo das regras esclarecidas com a Alexandra Borges (20/05/2026) sobre como apresentar faturas para receber o valor do prémio.

## Regras gerais

- O prémio **é acumulável** entre meses.
- O **excedente de faturas não transita** para meses seguintes — só conta para o(s) mês(es) ainda por liquidar.
- Recebe-se o **mínimo entre o prémio acumulado e o valor das faturas apresentadas**.
- Se não apresentar faturas num mês, **não recebe** esse mês (mas o prémio acumula para apresentar depois).
- O que ficar por justificar nesse ciclo, **perde-se**.

## Exemplo prático (dado pela Alexandra)

| Mês | Prémio | Faturas apresentadas | Recebido | Perdido |
|---|---|---|---|---|
| Janeiro | 100 € | 95 € | 95 € | 5 € |
| Fevereiro | 50 € | — | 0 € | — (acumula) |
| Março | 55 € | 110 € (Fev + Mar = 105 €) | 105 € | 5 € |

→ Os 5 € extra de Março **não transitam** para Abril.

## Tipos de despesa aceites

### Combustível
- **Gasóleo** → aceite.
- **Gasolina** → **não aceite**.
- A fatura tem de dizer "gasóleo". O que vai para o IRS é problema do contribuinte; aqui regista-se o que estiver escrito na fatura.

### Reparação automóvel / pneus
- Aceite **apenas se a fatura não identificar a viatura** (matrícula ou marca).
- Pneus normalmente não identificam → à partida aceites.
- Se a fatura tiver matrícula/marca diferente → **não aceite**.

### Material de escritório
- Papel, canetas, computadores, telemóveis.
- Sofá pequeno, secretária, bancos/cadeiras.

### Material de limpeza
- O que for.

### Transportes
- Táxi, Uber, passes.

### Outras
- Estadias de hotel.
- **Jantares até 60 € por recibo**.

## Notas importantes

- **A fatura manda**: se diz "combustível", regista-se como combustível; não vale reclassificar para "reparação auto" pelo CAE — isso é só para efeitos de IRS, não conta aqui.
- **Recibos verdes**: pendente — não confirmado se são aceites.

## Estratégia (notas pessoais)

- Garantir que faturas de reparação **não tenham matrícula** para serem aceites.
- Apresentar faturas em meses onde o acumulado de prémio é mais alto, para minimizar excedente perdido.
